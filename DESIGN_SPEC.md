# Ceetha Luxe – Design Specification

---

# Brand Identity

Theme: Luxury Minimalism  
Primary Colors:
- Black (#000000)
- White (#FFFFFF)
- Metallic Gold (#C6A75E)

Typography:
- Headings: Elegant serif (Playfair Display or similar)
- Body: Clean sans-serif (Inter / Open Sans)

---

# UI Principles

- High contrast
- Large product imagery
- Spacious layout
- Smooth hover animations
- Subtle gold accents (borders, buttons, highlights)
- Rounded but sharp luxury feel (8px border radius max)

# UX Requirements

- Fully responsive (mobile-first)
- Checkout ≤ 3 steps
- Payment feedback state
- Loading spinners for async calls
- Toast notifications for actions

---

# Animations

- Fade-in on scroll
- Button hover gold glow
- Smooth modal transitions
